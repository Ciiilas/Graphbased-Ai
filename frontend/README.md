# Frontend

React-Frontend von Graphbased-AI.

Hier entsteht das Web-Dashboard zur Visualisierung des Wissensgraphen, der Abhängigkeiten
und der KI-Antworten. Für die Graphdarstellung sind React Flow oder Cytoscape.js vorgesehen.

> **Status:** Platzhalter – die Implementierung befindet sich noch im Aufbau.

Node.js-Abhängigkeiten werden mit fester Version in `package.json` festgelegt.
